# 问题

光速那么快，为什么夜晚站在空旷的田野四周还是一片漆黑？

# 解答

奥伯斯佯谬
提出：1923年
该理论指出，**若宇宙是稳定的，无限且时空平直的，而且有无数个平均分布的发光形体，则无论望向天上哪一位置，都应该见到一粒星体的表面，星与星之间边不应有黑暗的位置。**
